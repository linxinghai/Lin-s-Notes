## Ndarray 对象

import numpy as np

```python
import numpy as np
a = np.array([1,2],[3,4])
print(a)
```

```python
import numpy as np
a = np.array([1,2,3,4,5],ndmin  = 2)
print(a)
```
>\[\[1 2 3 4 5\]\]
```python
import numpy as np
a = np.array([1,2,3],dtype = complex)
print(a)
```
